# Fortinet Licence Reader

Fortinet licences are typicaly delivered in multiple pdf files (one per licence) in side of multiple zip codes (one per SKU). This tool will extract all the licence codes inside these files making them easier to register. 

## Download and Install

Download the latest version from the releases page. 

To install just extract the .exe file anywhere on your computer

## Usage

1) Save .zip attachements for Fortinet Email containing your licence files into a folder
2) Choose Select Licence Folder in this tool
3) Browse and Select your folder
4) Press Submit and Enjoy your licence codes
